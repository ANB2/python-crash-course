The very important .gitignore

Your .gitignore file specifies files that should not be tracked.
You'll likely fill this with your files that are not inherently part of your project (autogenerated files, configuration files, etc.).
Git has a globbing syntax used for specifying what to ignore (see https://www.kernel.org/pub/software/scm/git/docs/gitignore.html for details).
